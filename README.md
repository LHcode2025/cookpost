# cookpost
## User Stories
User Story 1 : As a user I want to be able to post a recipe (must-have)

User Story 2 : As a user I want to be able to edit or delete recipes

User Story 3 : As a user I want to be able to update my recipes during the day.(must-have)

User Story 5 : As a user I want to get a warning notification, when I edit or delete something. (should-have) 

User Story 6 : As a user I want others to like or save my recipe. (could-have)
